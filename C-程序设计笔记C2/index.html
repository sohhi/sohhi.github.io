例子

  • <<：流插入运算符。要将后面的字符串插入到输出流中，需要向外部设备送信息时，要建立一个能够流通信息的通道。向显示器送信息的通道默认是打开的，而负责送数据的就是cout这个对象。送数据的动作用插入符实现。
  • endl：行结束
  • return 0；：返回函数计算结果。运行环境或者操作系统是主函数的调用者，默认情况下返回0。
  • #include <iostream>：iostream是一个文本文件的名字，给出了预定义好的输入输出的声明。include是一个编译预处理命令，在编译器开始做语法检查开始编译这段程序之前，首先要按照include指令到指定目录下找iostream这个文件，把这个文件的全部内容粘贴到include指令的位置。
  • using namespace std;：命名空间可以避免命名冲突。C++标准库里预定义好了很多功能模块（类、函数etc）。自己也会定义一些名字。所有标准库里的名字都有一个共同的命名空间，名字叫std，cout应该是std::cout（::表示作用域、分辨符）。using namespace std;表示打开std这个命名空间，这个命名空间里所有名就可以不带std这个姓了。

基本数据类型

整数类型

  • 基本整数类型：int
  • 按符号分：
    • signed
    • unsigned
  • 按照数据范围分
    • short
    • long
    • long long
      ISO C++标准并没有明确规定每种数据类型的字节数和取值范围，它只是规定它们之间的字节数大小顺序满足：
      （signed/unsigned）signed char
      <= （unsigned）short int
      <= （unsigned）int
      <= （unsigned）long int
      <= long long int
  • 字符类型（char）
    • 实质上存储的也是整数
    • 容纳单个字符的编码（8b）

实数（浮点数）类型

  • 单精度float
  • 双精度double
  • 扩展精度long double
    ISO C++标准并没有明确规定每种数据类型的字节数和取值范围，不同系统或编译环境会有不同实现

字符类型

  • 单字符char

浮点数常量

  • 1.25
  • 指数形式，例如0.345E+2 = 34.5，-34.4E-3 = -0.0344

浮点数在计算机中是近似存储的，比如3.14159存进去可能是3.1415899999999，所以不要在程序中比较两个浮点数。 比较浮点数一般是比较相减的差是否充分小。

变量

  • 在程序运行过程中允许改变的数据
  • 一个名字结合到一个内存单元
  • 需说明类型，这样编译器就会知道需要多少内存空间
  • 同时进行初始化是一个很好的习惯。若不进行初始化，则是无效数据，以前的垃圾数据。
    • int a = 0;
    • int a(0);
    • int a = {0};
    • int a{0};
      使用初始化列表的方式，不允许有数据丢失

数据丢失

比如用double值初始化int变量，至少小数部分会丢失，且double的表示范围更宽，整数部分也可能会有丢失。

赋值运算

C++中没有赋值语句，赋值是通过赋值表达式完成的。表达式的类型=赋值运算符左边对象的类型。

  • 复合的赋值运算符
    +=, *=, <<=等
    x *= y + 8 ⟹ x = x * (y + 8)

逗号运算

  • 格式
    表达式1, 表达式2, …,表达式n

  • 求解顺序及结果

    • 先求解表达式1，再求解表达式2, …
    • 最终结果为表达式n的值

  • 例1
    a = 3 * 5, a * 4 //最终结果为60

关系运算符

两个表达式的比较运算
<、<=、>、>=优先级相同（高）
==、!=优先级相同（低）
其结果为布尔类型

逻辑运算符

逻辑运算符：!(非) &&(与) ||(或)
优先级： 高 ⟹ 低
结果为布尔类型

  • 具有短路性质
    表达式1 && 表达式2：先求解表达式1，若为false，不再求解表达式2，得出结果false

条件运算符

表达式1 ? 表达式2 : 表达式3
x = a > b ? a : b

  • 条件运算符优先级高于赋值运算符，低于逻辑运算符。
  • a和b的类型可以不同。不论最终取a的值还是b的值，条件表达式的最终类型是a和b中较高的类型。

sizeof运算

  • C++并没有确切地规定占多少字节
  • 有些类里面会隐含地生成没有定义的数据成员，直接相加不一定准确
  • 语法形式
    sizeof(类型名)或sizeof 表达式
    sizeof(short)或sizeof x

位运算

高级语言处理数据一般以字节为单位，机器语言和汇编语言是以位为单位。但C语言诞生的时候是为了写操作系统，是高级语言中级别最低的。C++继承了C的位运算。

按位与运算（&）

  • 用途举例
    • 将某些位置0，其它位不变。
      例如a = a & 0xfe;将a的最低位置0。
    • 取出指定位
      例如c = a & 0xff;取出了a的低8位。

按位或运算（|）

  • 用途举例
    • 将某些位置1
      例如a = a | 0xff;将a的低字节置1。

按位异或运算（^）

  • 用途举例
    • 使得指定位翻转
      • 01111010
      • 00001111
      • 01110101

按位取反（～）

移位

  • 左移（<<）
    • 高位舍弃，低位补0
    • 相当于乘以2
  • 右移（>>）
    • 低位舍弃，无符号数高位补0，有符号数高位补符号位。
    • 相当于除以2
  • 不完全等同于乘除，溢出可能会导致 正数*2=负数
  • 在变量里存储了一些标记，有些运算需要把标记移一下位

类型转换

隐含转换

  • 一些二元运算符（算术运算符、关系运算符、逻辑运算符、位运算符和赋值运算符）要求两个操作数的类型一致
  • 在算术运算和关系运算中如果参与运算的操作数类型不一致，编译系统会自动对数据进行转换（即隐含转换），基本原则是将低类型数据转换为高类型数据

所谓低类型数据就是表示数据范围比较窄的数据
char | (unsigned)short | (unsigned)int | (unsigned)long | (unsigned)long long | float | double
低 ⟹ 高

  • 非布尔类型的算术值 ⟹ 布尔类型：算术值为0则false，否则true
  • 布尔值 ⟹ 非布尔类型：布尔值为false则0，布尔值为true则1
    （C中没有布尔类型，用0和1）
  • 浮点数 ⟹ 整数类型：保留整数部分，小数部分丢失
  • 整数值 ⟹ 浮点类型：小数部分记为0。如果整数所占空间超过了浮点类型的容量，精度可能有损失。

显式转换（强制类型转换）

  • 语法形式
    • 类型说明符(表达式)
    • (类型说明符)表达式
    • 类型转换操作符<类型说明符>(表达式)
      类型转换操作符：const_cast、dynamic_cast、reinterpret_cast、static_cast

例：int(z), (int)z, static_cast(z)三种完全等价
